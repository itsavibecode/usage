Settings

Email reorder reminders

Get a daily digest email when active products are approaching their average lifespan, so you can reorder before they run out.

  • Runs daily at 8am US Eastern (13:00 UTC).
  • One digest per email — at most one email per week so reminders don't repeat.
  • Skipped if nothing's due. You can turn this off any time.

Product data backfill

Some older products may be missing a brand or product image. This looks each one up in the UPC database and fills in what's missing — without changing anything you've already set.

Welcome to Usage Tracker

A simple log for the everyday products you use — toothpaste, shampoo, deodorant, razors, makeup, whatever runs out and needs replacing.

Why this exists

Household consumables get used up quietly. You only notice when you're standing at the bathroom mirror with an empty tube. This app makes the patterns visible:

  • Know when something's about to run out — before it does.
  • See what each product actually costs you per day.
  • Track which brands and types last longer for you.
  • Get a heads-up by email a few days before you'd be out.

Getting started in three steps

  1. Add a product. Tap + Add product in the top toolbar. Scan the UPC with your camera or type it in — brand name, product image, and product type all auto-fill from the UPC database.
  2. Track when you start using it. Either set a Start date when you add it, or leave it as Inventory and tap Start on the row later (one-tap, today's date pre-filled).
  3. Mark it finished when it runs out. Tap Finish on the active row. After a few finished products in a category, the app learns the average lifespan and starts surfacing reorder reminders for actives approaching that average.

Worth knowing

  • Stats are clickable. Tap Active / Inventory / Finished to filter the table to just those products. Tap Total spend / YTD / Last 30 days to see which products contribute to that total.
  • Email reminders. Open Settings (top-right) and toggle on. You'll get an email digest when products are running low, capped at one per week.
  • Bundles. Buying a 3-pack? Check Part of a bundle purchase and set the size — the app derives the per-item cost and tracks each sibling separately.
  • What's new. Click the version chip in the top-left any time to see recent changes to the app.
  • Try a demo at ?demo=1 — sample products, no sign-in needed, nothing saves.
  • Install as an app. On Chrome / Edge / Android, an Install app button appears in the toolbar — clicking it adds a home-screen icon and opens the app in its own window. On iPhone use Safari's share menu → Add to Home Screen.

Your data, privately

Products are stored in your own Firestore document under your Google account UID. No other user can see them. The only outbound calls are: the UPC database (when you add a product), brand-logo CDN, recall-alert API, and Resend if you opt in to email reminders. You can export everything to JSON or CSV any time from the Import/Export menu.
